PS4-OpenESPHost
PS4-OpenESPHost is an open-source firmware for the ESP8266 microcontroller.
This project enables or disables homebrews for PS4 under firmware 5.05.
The firmware is maintained by the community, you can propose your changes and report your problems via the GitHub : https://github.com/baptistecdr/PS4-OpenESPHost.
